Space Rocket

Your goal is to make the rocket ship fly the outer space.  

  1. Copy and paste the code below into a new Processing sketch:

                int x = 400;

int y = 600;

void setup() {

  background(0,0,40);

  size(800, 800);

}

void draw() {

  fill(random(255), 0, 0);

  ellipse(x, y + 130, 90, 90);      //Flame

  fill(248, 128, 0);

  ellipse(x, y + 115, 70, 70);      //Flame

  fill(255, 153, 0);

  ellipse(x, y + 95, 35, 35);       //Flame

  fill(100, 100, 100);

  triangle(x, y + 10, x + 50, y + 100, x - 50, y + 100);  //Rocket

}
